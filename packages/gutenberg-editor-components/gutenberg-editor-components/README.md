# Gutenberg editor components

This package includes several UI components to enhance the InspectorControls.

## Install

```JS
npm install npm install @yardinternet/gutenberg-editor-components --save
```

## Usage

```JS
import { FlexControl from '@yardinternet/gutenberg-editor-components' }
```

## Hooks

Available hooks

### SpacingControl

```JS
addFilter(
	'gutenberg-editor-components.spacingSize',
	'gutenberg-editor-components',
	() => 7
);
```
